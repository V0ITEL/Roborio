1. Overview

Roborio is the first decentralized marketplace for robot rentals built on Solana blockchain. We connect robot operators with businesses needing automation, enabling flexible, on-demand access to robotic services.

Key Features:
  • Flexible pricing: per-task, per-minute, or per-kilometer
  • Multiple payment options: SOL, USDC, or $ROBORIO (20% discount)
  • Smart contract escrow for secure transactions
  • Real-time tracking and monitoring
  • Transparent rating system

2. Getting Started

For Businesses (Renters)

  1. Connect Your Wallet - Use Phantom, Solflare, or any Solana-compatible wallet
  2. Browse Robots - Filter by type, location, pricing, and availability
  3. Select & Book - Choose your robot and rental duration
  4. Pay & Deploy - Payment is held in escrow until task completion
  5. Track & Verify - Monitor robot performance in real-time
  6. Rate & Review - Help others by leaving feedback

For Operators (Providers)

  1. Register Your Robot - Submit robot specifications and certifications
  2. Set Pricing - Choose per-task, per-minute, or per-km pricing
  3. List on Marketplace - Make your robot available for rent
  4. Accept Bookings - Receive automated notifications
  5. Complete Tasks - Robot performs the work
  6. Get Paid - Automatic settlement via smart contracts

3. How It Works

The Three-Layer Architecture

Layer 1: Physical Robots

  • Delivery robots for last-mile logistics
  • Cleaning robots for commercial spaces
  • Security robots for patrol and monitoring
  • Inspection drones for infrastructure
  • Warehouse AGVs for automation

Layer 2: Pricing Engine

  • Per-task: Fixed price for specific jobs
  • Per-minute: Time-based rental
  • Per-km: Distance-based pricing

Layer 3: Blockchain Platform

  • Smart contracts for escrow and payments
  • Real-time tracking via IoT integration
  • Transparent rating and review system
  • REST API for third-party integration

4. Pricing Models

Per-Task Pricing

Fixed price for completing a specific job. Best for: package delivery, security patrols, one-time cleaning.

Example: Deliver 10 packages = 5 SOL flat fee

Per-Minute Pricing

Pay only for the time the robot is actively working. Best for: flexible tasks, variable durations.

Example: Floor cleaning @ 0.1 SOL/minute

Per-Kilometer Pricing

Pay based on distance traveled. Best for: delivery, patrol routes, inspection.

Example: Security patrol @ 2 SOL/km
Payment Discount: Get 20% off all rentals when paying with $ROBORIO tokens instead of SOL or USDC.

Roborio is a decentralized marketplace built on Solana where businesses can rent robots on-demand. Think of it as "Uber for robots."

How do payments work?

Payments are handled through Solana smart contracts. Funds are held in escrow and released to the operator once the task is completed and verified.

Delivery robots, cleaning robots, security robots, inspection drones, and warehouse automation systems.
